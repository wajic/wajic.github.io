Welcome to WAjic - WebAssembly JavaScript Interface Creator

WAjic is a simple way to build C/C++ WebAssembly application with meaningful browser integrations like WebGL.

Samples

See the samples page for some demos and source code of how WAjic can be used

Getting started

Check out the GitHub repository README for a detailed description, features, usage and how to get started.

WAjicUp

Go to WAjicUp to access the WebAssembly JavaScript Interface Creator Utility Program.

Viewer

Use the WAjic viewer to load and test your WASM files immediatly after building them.
